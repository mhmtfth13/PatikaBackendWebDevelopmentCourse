# ASP.NET Core MVC Project - Netflix Clone

Bu proje, ASP.NET Core MVC kullanılarak geliştirilmiş basit bir Netflix benzeri web uygulamasıdır.

## Proje Yapısı
```
ProjectName/
├── Controllers/
│   └── HomeController.cs
├── Models/
├── Views/
│   ├── Home/
│   │   └── Index.cshtml
│   └── Shared/
│       └── _Layout.cshtml
├── wwwroot/
│   ├── css/
│   ├── js/
│   └── images/
└── Program.cs
```
